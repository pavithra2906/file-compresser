# Web-Based File Compression Utility

This is a web-based file compression and decompression utility that runs entirely in the browser and on the server.

## Features
- Compress files using GZIP or ZIP format
- Decompress .gz and .zip files
- Web interface accessible from any device
- Server-side processing for security

## Access
Open http://localhost:8080 in your web browser after starting the server.

## Usage
1. Navigate to the Compress or Decompress page
2. Select a file
3. Click the action button
4. Download the result

## Technical Details
- Pure Java backend
- Server-side rendering
- No JavaScript required for core functionality
- Mobile-responsive design
